## 실행방법
```
# 1. 가상 환경 생성
python3 -m venv venv

# 2. 가상 환경 활성화
source venv/bin/activate  # macOS/Linux

# 3. pip 최신 버전 업데이트
pip install --upgrade pip

# 4. pygame 설치
pip install pygame

# 5. 실행
python3 Worm_Game.py
```
